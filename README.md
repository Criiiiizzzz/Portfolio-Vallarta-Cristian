Portfolio – Cristian Vallarta

Este repositorio concentra proyectos de desarrollo de software orientados principalmente a aplicaciones web y plataformas de comercio electrónico. Los proyectos incluidos reflejan experiencia en diseño e implementación de soluciones web con arquitectura basada en roles, integración con bases de datos relacionales y estructuración modular del código.

Entre los trabajos más relevantes se encuentra Mercadito, plataforma tipo marketplace desarrollada con JSP y Java, reconocida con el Segundo Lugar en el 32° Concurso Nacional “Premio a los Mejores Prototipos” del Instituto Politécnico Nacional. El sistema contempla gestión de usuarios con distintos roles (administrador, vendedor y usuario), administración de catálogo de productos, control de sesiones e integración con base de datos MySQL.

Las tecnologías empleadas en los distintos proyectos incluyen Java, JSP, HTML, CSS, JavaScript, MySQL, Maven y control de versiones con Git.

Este portafolio reúne los proyectos más relevantes de mi trayectoria académica y evidencia el desarrollo de mis competencias técnicas en el área de software.
