Reacciones quimicas inorgánicas

⊱ Química inorgánica ⊰

Proceso químico en el cual dos o más sustancias denominadas como “reactivos” por causa de un factor energético se transforman en nuevas sustancias, a las cuales se les denomina como “productos”, dichas sustancias obtenidas varían según las condiciones a las cuales se les someta a los reactivos de la reacción. Este proceso se le representa por medio de una ecuación química:





reactivos -> productos




✧ Tipos de reacciones:


1. Reacciones de síntesis:
Las reacciones de síntesis consisten en la unión de compuestos sencillos para la formación de compuestos más complejos, las reacciones de síntesis siguen esta estructura:

A + B → AB



Siendo ejemplo de esto la siguiente reacción:




Na + Cl → NaCl




2. Reacciones de descomposición:



En las reacciones de descomposición un compuesto se fragmenta en elementos o compuestos más sencillos , la representación de dichas reacciones es la siguiente:



AB → A+B



Ejemplo de este tipo de reacciones es:



NaCl → Na + Cl



3. Reacciones de desplazamiento simple:



En este tipo de reacciones uno de los elementos pertenecientes a un compuesto remplaza la posición de otro de elemento dentro de un compuesto diferente, el modelo de dicha reacción es el siguiente:
