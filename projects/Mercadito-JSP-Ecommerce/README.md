
### Mercadito – Plataforma E-commerce con JSP (Segundo Lugar)

Prototipo académico desarrollado para el 32° Concurso Nacional “Premio a los Mejores Prototipos del Nivel Medio Superior” del Instituto Politécnico Nacional, donde obtuvo el Segundo Lugar en la categoría Aplicación para la Empresa.

Desarrollo de plataforma web tipo marketplace con arquitectura basada en roles (administrador, vendedor y usuario), gestión de catálogo de productos, control de sesiones e integración con base de datos relacional MySQL.

Tecnologías utilizadas: Java, JSP, HTML, CSS, JavaScript, MySQL y Maven.
