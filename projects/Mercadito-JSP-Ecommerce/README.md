
### Mercadito – Plataforma Web E-commerce con JSP - 2023

Mercatec es una plataforma web de comercio electrónico desarrollada con Java y JSP, orientada a la digitalización de mercados locales mediante un modelo tipo marketplace.

El sistema permite la gestión integral de usuarios y productos, incorporando una arquitectura basada en roles (administrador, vendedor y usuario) y una estructura modular organizada con Maven.

Características

Gestión de usuarios con distintos roles y permisos.
Administración de catálogo de productos.
Control de sesiones y autenticación.
Integración con base de datos relacional.
Arquitectura estructurada bajo estándar Java EE.
Organización modular del proyecto.

Tecnologías utilizadas

Java
JSP
HTML
CSS
JavaScript
Maven
MySQL
Git

Reconocimiento

Proyecto galardonado con el Segundo Lugar en el 32° Concurso Nacional “Premio a los Mejores Prototipos” del Instituto Politécnico Nacional, en la categoría Aplicación para la Empresa.

Objetivo

Desarrollar una solución web dinámica orientada a comercio electrónico, aplicando principios de arquitectura basada en roles, integración con bases de datos relacionales y estructuración modular del código.
